## CPU的组成

CPU（中央处理器）核心组成可概括为「运算器 + 控制器 + 寄存器组 + 内部总线」，以下为各组成模块的详细对比

| CPU组成模块 | 核心部件           | 核心功能                              | 关键组成/细节（软考高频）                                                                                           |
| ------- | -------------- | --------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 运算器     | ALU（算术逻辑单元）    | 完成数据的算术运算和逻辑运算，执行数据处理             | - 累加寄存器（AC）：存放操作数、中间/最终结果- 数据缓冲寄存器（DR）：暂存内存/外设数据，协调速度差异- 状态条件寄存器（PSW）：存放状态标志位（CF、OF等），为分支指令提供判断依据       |
| 控制器     | CU（时序控制部件）     | 作为“指挥中心”，指挥协调各部件，控制指令的读取、译码和执行    | - 程序计数器（PC）：存放下一条指令地址，自动递增- 指令寄存器（IR）：暂存当前执行指令- 指令译码器（ID）：解析指令的操作码和操作数地址- 地址寄存器（AR）：存放要访问的内存单元地址，用于地址定位 |
| 寄存器组    | 无单一核心，分通用/专用两类 | 高速临时存储数据、地址和指令，减少CPU与内存交互，提升运行效率  | - 通用寄存器：灵活暂存数据/地址（如x86架构的AX、BX等）- 专用寄存器：功能固定（如PC、IR、AR、DR、PSW等），是CPU正常工作的必要组件                           |
| 内部总线    | 无核心部件，分三类总线    | 作为CPU内部各部件的“通道”，实现数据、地址和控制信号的快速传输 | - 数据总线：传输数据- 地址总线：传输内存单元地址- 控制总线：传输控制信号，保障各部件协同工作                                                       |
